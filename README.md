ユーザー側の仕様

・ユーザーログイン
・日時と時刻のリアルタイム表示 
・出勤、退勤、休憩、戻りの打刻 
・月の残業時間の表示
（・ログアウト）

管理者側の仕様

・ユーザー情報編集
・従業員の名前から打刻時間の確認と編集 
・従業員ごとの就業時間、残業時間の確認確認（指定期間ごと）
・残業アラート（一定の残業時間を超えた社員をアラートでお知らせ）

データベースの仕様

勤怠
LユーザーID(ユーザー１のキーと紐づいているレコード)
L日付
L出勤時間
L休憩時間
L休憩戻り
L退勤時間
